🧮 尿蛋白/クレアチニン比（随時尿での蛋白尿評価）

尿蛋白/Cr比（uPCR）：
= 尿蛋白濃度(mg/dL) ÷ 尿クレアチニン濃度(mg/dL)

uPCRのカットオフ：
< 0.15 g/gCr → A1（正常）
0.15〜0.49 g/gCr → A2（中等度蛋白尿）
≥ 0.50 g/gCr → A3（高度蛋白尿）

本例：0.40 → A2区分（中等度蛋白尿）

🔬 ネフローゼ症候群をきたす疾患

ａ アミロイド腎症 → ネフローゼ症候群：
アミロイド繊維が糸球体基底膜・メサンギウムに沈着
→ 基底膜の透過性亢進 → 大量蛋白尿（≥3.5g/日）→ ネフローゼ

他の選択肢がネフローゼを呈さない理由：
ｂ 良性腎硬化症 → 高血圧による糸球体硬化。微量〜少量蛋白尿（ネフローゼには至らない）
ｃ 慢性間質性腎炎 → 尿細管間質病変。蛋白尿は少量・低分子蛋白（β2-MG）が主体
ｄ Bartter症候群 → TA-NKCC2障害 → 低K血症・代謝性アルカローシス。糸球体は正常
ｅ Liddle症候群 → ENaC過活性 → 高血圧・低K血症。糸球体病変なし

🔬 巣状分節性糸球体硬化症（FSGS）の特徴

診断根拠：
・「一部の糸球体に同様の所見」→ 巣状（focal）
・分節性（segmental）硬化
・蛍光IgM陽性（硬化部位への非特異的IgM沈着）
・血尿＋蛋白尿（ネフローゼ症候群）

ｃ IgGクリアランスが増加：
FSGS = 非選択性蛋白尿（足突起消失＋基底膜損傷で大分子も通過）
→ IgG（分子量160kDa）も濾過される → クリアランス増加

誤り選択肢：
ａ 小児慢性糸球体腎炎で最多 → IgA腎症が最多（FSGSではない）
ｂ 血清C3が低下 → FSGSでは補体正常
ｄ ステロイドが著効 → FSGSはステロイド抵抗性が多い
ｅ 予後は良好 → FSGSは予後不良（末期腎不全に至ることが多い）

🎯 国試ポイント

FSGS = 非選択性蛋白尿（IgGクリアランス↑）＋ステロイド抵抗性＋予後不良
小児慢性糸球体腎炎の最多 = IgA腎症（FSGSではない）
微小変化群との違い：FSGS = 非選択性・ステロイド抵抗性・光顕で硬化所見あり

🔬 顕微鏡型多発血管炎（MPA）の診断根拠

RPGN（急速進行性糸球体腎炎）のパターン：
・高齢男性（ANCA関連血管炎は高齢者に多い）
・感冒後→急速なCr上昇（4.0）
・腎萎縮なし（急性変化→AKI、慢性なら萎縮あり）
・血尿2+＋蛋白尿（糸球体炎症）
・PAS染色：細胞性半月体形成（RPGN の組織所見）

顕微鏡型多発血管炎（MPA）：
・ANCA関連血管炎（MPO-ANCA陽性が多い）
・pauci-immune型（免疫複合体沈着なし）
・治療：ステロイドパルス＋シクロホスファミド

他の選択肢が誤りの理由：
ａ FSGS → 慢性経過・補体正常・半月体なし
ｂ Alport症候群 → 遺伝性・若年発症・難聴・眼症状
ｄ 糖尿病性腎症 → 血糖正常（90mg/dl）で除外
ｅ 骨髄腫腎 → 蛋白尿はBJ蛋白（κ/λ軽鎖）、腎生検で円柱腎症
