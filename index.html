☁️ 進捗の複数デバイス同期
GitHub Gist を使って進捗をクラウドに保存します。
PC・iPad・スマホ間で学習記録が共有されます。
初回セットアップ
1. GitHub → Settings → Developer settings → Personal access tokens を開く
2. スコープ「gist」だけにチェックしてトークンを生成
3. 生成されたトークンを下に貼り付けて「保存して同期」をタップ
4. 別のデバイスでも同じトークンと Gist ID を入力すれば同期開始
