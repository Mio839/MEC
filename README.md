# 国試学習ツール

医師国試 臓器別問題集 | HTML Viewer Q（iPad）・ブラウザ（PC）対応

## 使い方

### オンライン

https://[ユーザー名].github.io/[リポジトリ名]/

### ローカル（PC）

`index.html` をブラウザで直接開く

## 機能

- ✅ チェックボックス（解いた問題を記録・localStorage保存）
- 📊 章別進捗バー
- 🔍 キーワード検索
- 🔖 フィルタ（難問/標準/易問/必修/未チェック/済）
- 📝 メモ機能（問題ごとにメモ保存）
- 章の折りたたみ
- 解答の一括開閉

## 新しい分野を追加するには

1. 追加したいHTMLファイルをClaudeのプロジェクトにアップロード
1. Claudeに以下を送る：

```
以下のHTMLを追加して index.html を再ビルドしてください。
- [ファイル名]：[分野名]（第X章〜第X章）
```

1. 生成された `index.html` をこのリポジトリにアップして上書き

## HTMLの仕様（追加ファイルの要件）

追加するHTMLが以下の仕様を満たしていること：

|要素   |仕様                      |
|-----|------------------------|
|問題カード|`class="qc"` `id="q{n}"`|
|正答率  |`data-rate="{数字}"`      |
|問題文  |`class="qt"`            |
|正解選択肢|`class="ch2 ok"`        |
|解説   |`class="eg"`            |
